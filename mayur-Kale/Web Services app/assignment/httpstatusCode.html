Http Status Codes

  • 200 OK - Request has succeeded
  • 400 Bad Request - Request could not be understood by the server due to malformed syntax
  • 401 Unauthorized - The request requires user authentication
  • 404 Not Found - The server has not found anything matching the Request-URI
  • 500 Internal Server Error - The server encountered an unexpected condition which prevented it from fulfilling the request.
  • 503 Service Unavailable - The server is currently unable to handle the request due to a temporary overloading or maintenance of the server
  • 408 Request Timeout - The client did not produce a request within the time that the server was prepared to wait
  • 502 Bad Gateway - The server was acting as a gateway or proxy and received an invalid response from the upstream server
  • 403 Forbidden - Error code for user not authorized to perform the operation or the resource is unavailable for some reason
  • 501 Not Implemented - The server does not support the functionality required to fulfill the request
